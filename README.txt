================================================================================
                    面向对象图表生成系统（Qt Designer版）
                    Object-Oriented Chart Generation System — Qt6 / C++17
================================================================================

一、项目简介
--------------------------------------------------------------------------------

本项目是一个基于 Qt 6 + Qt Designer 的桌面图表生成工具，采用面向对象多态架构设计。
系统定义了一个抽象图表基类 Chart，通过 Template Method 模式将 QPainter 初始化与
具体绘制解耦，派生出柱状图（Histogram）、折线图（LineChart）和饼图（PieChart）
三种图表类型。三种图表共享一个 DataSource 数据源，数据变更时通过 Qt 信号-槽机制
自动刷新所有关联图表，实现了典型的观察者模式。

用户界面采用 Qt Designer .ui 表单构建静态布局，通过 MainWindow 类管理动态行为，
实现了界面与逻辑的分离，可直接在 Qt Designer 中可视化编辑窗口布局。


二、运行环境与依赖
--------------------------------------------------------------------------------

1. 操作系统
   - Windows 10/11（64 位）
   - Linux
   - macOS

2. 编译工具链
   - MinGW 13.1.0（Windows 推荐）或 MSVC 2019+
   - GCC 8+ / Clang 10+（Linux / macOS）
   - 编译器需支持 C++17 标准

3. Qt 框架
   - Qt 6.x（推荐）或 Qt 5.x（源码含 Qt5 兼容性预处理宏）
   - 必需模块：Qt Widgets

4. 构建系统
   - qmake（Qt 自带）

5. 无第三方库依赖
   本项目仅依赖 Qt 框架和 C++ 标准库，无额外第三方库。


三、构建与运行
--------------------------------------------------------------------------------

1. 确保 Qt 开发环境已正确安装，qmake 和编译器路径已加入系统 PATH 环境变量。

2. 生成 Makefile：
   qmake chart-project.pro

3. 编译：
   make          （Linux / macOS，或使用 mingw32-make / nmake 等对应工具）

4. 运行生成的可执行文件。

5. 清理重新编译：
   make clean && make


四、软件功能
--------------------------------------------------------------------------------

1. 三种图表类型，一键切换
   - 柱状图（Histogram）：以圆角矩形柱体展示各数据项的值，柱体颜色随数值
     大小渐变，柱顶标注数值，底部标注数据标签或序号。
   - 折线图（LineChart）：以折线连接各数据点，底部带半透明区域填充，每个
     数据点用圆点标记并标注数值。
   - 饼图（PieChart）：按数据占比绘制扇形，各扇区使用 HSL 色彩空间均匀分布
     的颜色，扇区内标注百分比，右侧附带图例和合计信息。

2. 数据管理
   - 从文件加载纯数值数据：支持 .txt、.csv、.dat 等文本格式，文件中数字以逗号、
     空格或换行符分隔即可。
   - 从文件加载带标签数据：支持 "标签,数值" 格式的 CSV / 文本文件。标签作为
     X 轴或图例文字显示，使图表具有实际含义（如产品名称对应销售额）。
     自动检测并跳过首行标题头，支持逗号、制表符、空格三种分隔符。
   - 随机生成数据：弹出对话框输入数据个数（1~50），自动在 10~100 范围内
     生成随机数值（保留一位小数）。
   - 清空数据：一键清空当前所有数据。

3. 实时状态栏
   窗口底部状态栏实时显示：数据条数（含是否标签数据标识）、合计、最小值、最大值。
   数据变更时自动刷新。

4. 自动刷新
   数据源变更（加载、生成、清空）→ 发出 dataChanged 信号 → 图表自动重绘、
   状态栏自动刷新，无需手动触发。


五、操作流程
--------------------------------------------------------------------------------

1. 启动程序，默认显示柱状图，并预置了 10 个初始数据。
2. 点击工具栏按钮选择数据来源：
   - "加载文件"     → 在文件对话框中选择纯数值数据文件 → 图表自动刷新
   - "加载标签文件" → 选择 "标签,数值" 格式的 CSV 文件 → 图表自动刷新（标签显示于坐标轴/图例）
   - "随机生成"     → 输入期望的数据个数 → 图表自动刷新
   - "清空数据"     → 清空所有数据，图表显示空数据提示
3. 点击图表类型按钮（柱状图 / 折线图 / 饼图）切换显示方式，
   当前激活的图表按钮以蓝色高亮显示。
4. 观察底部状态栏，了解当前数据的统计信息。


六、输入输出格式
--------------------------------------------------------------------------------

1. 数据文件输入格式

   A. 纯数值文件（"加载文件"按钮）
      - 文件类型：纯文本文件（.txt / .csv / .dat 均可）
      - 编码：UTF-8
      - 分隔符：逗号 (,)、空格、换行符均可混合使用
      - 数据格式：每项为可解析为浮点数的文本
      - 示例文件内容：
          25, 48, 32, 60, 42
          55 38 70 29 51
        或
          12.5
          30.8
          45.2
      - 解析逻辑：程序读取整个文件内容，使用正则表达式拆分 token，
        每个 token 尝试转换为 double，成功则纳入数据集。

   B. 标签数据文件（"加载标签文件"按钮）
      - 文件类型：CSV 或纯文本文件（.csv / .txt）
      - 编码：UTF-8
      - 格式：每行一条记录，包含标签和数值，例如：
          产品名称,销售额
          手机,125.5
          笔记本电脑,89.3
          平板电脑,66.8
      - 分隔符支持：逗号 (,)、制表符 (Tab)、多个空格均可
      - 标题头自动检测：若第一行第二列为非数值文本，则视为标题头自动跳过
      - 标签可包含空格（如 "笔记本电脑"），超长标签在图表中自动截断加省略号
      - 解析失败提示：若文件中无有效 "标签,数值" 记录，弹出错误提示

2. 随机生成数据格式
   - 选择"随机生成"后弹出对话框，输入数据个数 N（1 ≤ N ≤ 50）
   - 系统在 [10.0, 100.0) 范围内生成 N 个随机值，保留一位小数
   - 生成后直接替换当前数据集（同时清除标签状态）

3. 图表输出格式
   - 柱状图：X 轴优先显示数据标签（有标签数据时），否则显示序号（1, 2, 3…）；
     Y 轴为数值；柱顶标注精确到一位小数的数值
   - 折线图：X 轴优先显示数据标签，否则显示序号；Y 轴为数值；
     数据点为实心圆，折线上方标注数值
   - 饼图：扇区按数据占比分配角度，扇区内标注百分比（如 "35.2%"），
     右侧图例优先显示数据标签（如 "手机: 125.5 (23.0%)"），否则显示
     "数据 N: 值 (百分比%)"，底部附带合计信息

4. 状态栏输出格式
   数据条数: N [标签]  │  合计: XXX.X  │  最小值: XX.X  │  最大值: XX.X
   （有标签数据时显示 "[标签]" 标识；无数据时显示 "—"）


七、项目架构说明
--------------------------------------------------------------------------------

架构图：

  DataSource (QObject)           ← 数据存储（数值+标签）、文件读取、随机生成、信号发射
      │  raw pointer
      ▼
  Chart (abstract QWidget)       ← Template Method: paintEvent() → virtual draw()
      ├── Histogram              ← 柱状图：圆角矩形柱体 + 渐变色
      ├── LineChart              ← 折线图：折线 + 面积填充 + 数据点
      └── PieChart               ← 饼图：HSL 扇区 + 图例 + 中心装饰

  MainWindow (QMainWindow)       ← Qt Designer .ui 表单 + 信号槽逻辑
      └── mainwindow.ui          ← 静态布局（可 Qt Designer 可视化编辑）

文件结构：

  chart-project.pro              — qmake 工程文件 (Qt6, C++17, widgets 模块)
  mainwindow.ui                  — Qt Designer 表单：主窗口静态布局
  mainwindow.h                   — MainWindow 类声明 (QMainWindow 子类)
  mainwindow.cpp                 — 信号槽连接、图表切换、状态栏刷新、数据操作
  main.cpp                       — 程序入口：QApplication、全局样式表
  datasource.h / datasource.cpp  — DataSource 数据源类（含标签数据支持）
  chart.h / chart.cpp            — Chart 抽象基类：paintEvent、坐标轴、网格、标题
  histogram.h / histogram.cpp    — Histogram 柱状图绘制
  linechart.h / linechart.cpp    — LineChart 折线图绘制
  piechart.h / piechart.cpp      — PieChart 饼图绘制
  sample-sales.csv               — 示例标签数据文件（产品销售额）
  sample-scores.csv               — 示例标签数据文件（学生成绩）

设计模式：

  - Template Method：Chart::paintEvent() → 子类 draw()
  - Observer（Qt 信号-槽）：DataSource::dataChanged → Chart::update() + 状态栏刷新
  - 多态：Chart* 指针统一管理不同图表子类实例
  - UI/逻辑分离：.ui 表单定义静态布局，MainWindow 管理动态行为
